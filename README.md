# CheckStatus

ボットのステータスを確認するためのプラグインです。

## ✨ 特徴
- ボットのシステム情報（CPU, RAM）を取得可能
- ボットの稼働状況（Ping, サーバー数, 稼働時間）を取得可能
- 丸型ブロックとして提供されるため、メッセージ送信や埋め込みブロックと組み合わせて使用可能

## 📦 追加されるブロック
- **💻 CPU使用率 (%)**
- **🧠 RAM使用量 (GiB)**
- **📊 RAM使用率 (%)**
- **📡 Ping (ms)**
- **⏱️ 現稼働時間**
- **🏠 サーバー数**
- **🛠️ コマンド数**

> [!IMPORTANT]
> このプラグインを使用するには、Python環境に `psutil` ライブラリがインストールされている必要があります。
> インストールされていない場合は、ターミナルで `pip install psutil` を実行してください。

## 🚀 インストール方法

1.  [Easy Discord Bot Builder](https://himais0giiiin.com/beta/) を開きます。
2.  「プラグイン」ボタンをクリックします。
3.  「GitHubで探す」タブで `CheckStatus` を検索するか、以下のインストールURLをブラウザに貼り付けます：
    `https://himais0giiiin.com/beta/?install-plugin=yuta25912/CheckStatus`

## 🛠️ 開発者向け

このプラグインをローカルでテストするには、リポジトリを ZIP 形式でダウンロードし、EDBP のプラグイン画面から「ZIPからインストール」を選択してください。

### ファイル構成
- `manifest.json`: プラグインの定義
- `plugin.js`: ブロックの実装とロジック
